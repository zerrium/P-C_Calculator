# Permutation and Combination Calculator (GUI)
Developed with Gtk# 2.12.45 for the GUI and Mono 6.12.0.90 for the C# compiler and cross-platform compatibility

## Compatibility:
* Linux (Tested on Ubuntu 20.04.1 LTS, Recommended)
* Windows (Tested on Windows 10)
* MacOS (Theoretically, untested)

## Installation:
* Linux:
  1. Install Mono [here](https://www.mono-project.com/download/stable/#download-lin) and on the step 2 use ```mono-complete``` instead of ```mono-devel```
  2. Verify the installation by typing ```mono --version``` on Terminal
  3. Double click the .exe file or right mouse click at it and choose: Open With Mono Runtime
  4. (Optional) If you want to see the debug messages, use Terminal at the .exe file location and type ```mono filename.exe``` command
  5. It might take a while to launch the application, just wait and it will appear
* Windows:
  1. Install Gtk# for .NET [here](https://www.mono-project.com/download/stable/#download-win) (the second row)
  2. Double click the .exe file
  3. (Optional) If you want to see the debug messages, open CMD at the .exe file location and type ```filename.exe >> output.txt``` command
  4. It might take a while to launch the application, just wait and it will appear
* MacOS: unknown, possibly follow [this intruction](https://www.mono-project.com/download/stable/#download-mac)

# Permutation and Combination Calculator (CLI)
Developed with Mono 6.12.0.90 for the C# compiler on Ubuntu

## Compatibility:
* Linux (Tested on Ubuntu 20.04.1 LTS, Recommended)
* Windows (Tested on Windows 10)
* MacOS (Theoretically, untested)

## Installation:
* Linux:
  1. Install Mono [here](https://www.mono-project.com/download/stable/#download-lin) and on the step 2 use ```mono-complete``` instead of ```mono-devel```
  2. Verify the installation by typing ```mono --version``` on Terminal
  3. Open Terminal at the .exe file location
  4. Type ```mono filename.exe``` command, optionally if you want to see the debug message use ```mono filename.exe debug``` command instead
* Windows:
  1. Double click the .exe file
  2. (Optional) if you want to see the debug message, open CMD at the file location and use ```filename.exe debug``` command
* MacOS: unknown, possibly follow [this intruction](https://www.mono-project.com/download/stable/#download-mac)
